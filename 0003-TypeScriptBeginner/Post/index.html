Coding Challenge:
Convert this side of this app into a TODO app,
Requirements:

  • Use TYPESCRIPT, all function must be type strict for params and return value where applicable!
  • Use 3 built-in types.
  • Use at least 1 custom type.
  • Use at least 1 custom class, with at least 1 function and constructor
  • User must be able to add new TODO items.
  • User must be able to check off list items (There must be some way to indicate that task is done.)
Stretch Goal:
  • Make TODO items removable.
  • Make TODO items editable.
  • Make delete trash can icon work to remove notes for NOTES side of APP.
